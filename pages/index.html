Welcome to clib+'s Website! clib+ is brand new C package extending the standard libs that comes with just about any linux OS designed to abstract most low-level functionalities to a high-level standard similar to higher-level languages.

The most important one of all. The String Library. Abstracting all memory management in the library so user do not need to deal with memory management beside freeing data!
